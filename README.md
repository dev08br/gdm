# GDM

Sistema de processamento e governança documental estruturada para conversão de documentos legados em grafos de conhecimento semântico e exportação no formato GovDoc Model (GDM).

## 🚀 Proposta e Paradigma
Este projeto visa quebrar a dependência de formatos tradicionais de escritório (.docx/.odt), tratando o documento como um objeto de dados e registro digital estruturado, auditável e computável.

## 🛠️ Stack Tecnológica Mínima
* **Core & Parser:** Python (xml.etree.ElementTree & python-docx)
* **Visualização:** Pyvis (HTML Interativo)
* **Infraestrutura:** Docker & Docker Compose
